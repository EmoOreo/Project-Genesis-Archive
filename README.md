# Project Genesis: Archive

**Project Genesis: Archive** is a browser-based idle collection game about discovering, registering, and archiving life across deep time.

## v0.2.0 Expedition & Discovery

This release expands the v0.1.0 foundation into a more complete playable slice.

### Added

- 25 starter species records
- 25 Archive-style species portrait SVGs
- 6 habitat nodes
- Habitat selection and habitat unlocks
- Discovery registration by habitat
- Rarity-weighted discoveries
- Collection progress tracking
- Improved Archive cards with images
- Updated local save key for v0.2.0

### Notes

The included visuals are lightweight generated SVG archive portraits, not final sourced wildlife photos or paleoart. They are placeholders for the image pipeline so GitHub Pages can run the game without external assets.

## Run

Open `index.html` through GitHub Pages or a local/static web server. Browser security may block JSON loading when opening directly from a file path.
